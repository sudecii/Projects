# Сегментация клиентов онлайн-магазина подарков

## Описание проекта
Проект посвящён анализу данных онлайн-магазина и сегментации клиентов с помощью RFM-анализа и методов кластеризации.  
Цель работы — выделить группы покупателей для последующего применения персонализированных маркетинговых стратегий.

## Этапы проекта
- Загрузка и предобработка данных  
- Очистка данных (пропуски, дубликаты, возвраты, нулевая стоимость)  
- Разведывательный анализ (EDA)  
- Построение RFM-таблицы  
- Кластеризация клиентов (KMeans)  
- Интерпретация и визуализация сегментов  
- Выводы и рекомендации  

## Результаты
В результате анализа были выделены несколько сегментов клиентов:  
- Лояльные — часто покупают и приносят основную часть выручки  
- Перспективные — проявляли активность недавно, но покупают нерегулярно  
- Спящие — давно не совершали покупок, требуют стимулирования  

Эти сегменты позволяют адаптировать маркетинговые кампании под разные группы клиентов.
